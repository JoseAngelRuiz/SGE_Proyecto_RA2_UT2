# 06 — Configuración inicial

* Localiza archivos de **configuración** y el puerto por defecto.
  El archivo de configuracion de nuestro Odoo esta en la siguiente ruta.
  **C:\Program Files\Odoo 19.0.20251111\server**

Ahi podremos ir al archivo odoo.conf, dentro de ese archivo podemos configuarr  la contraseña a la base de datos el host, el puerto, el usuario y la contraseña de administrador.

* Anota la **URL de acceso local**

[localhost:8069](http://localhost:8069/)
